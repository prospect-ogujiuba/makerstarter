Makerstarter WordPress Theme for Creative Developers

Start building a Block-based WordPress theme in minutes. No bloat, just the essentials. 


# Install Makerstarter theme
$ git clone https://github.com/prospect-ogujiuba/makerstarter.git
$ wp theme activate makerstarter

# Install Makerblocks plugin
$ git clone https://github.com/prospect-ogujiuba/makerblocks.git
$ wp plugin activate makerblocks

# Start development
$ cd wp-content/themes/makerstarter
$ npm install && npm run dev
$ cd wp-content/plugins/makerblocks
$ npm install && npm run dev
